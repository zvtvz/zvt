ZVT是在[fooltrader](https://github.com/foolcage/fooltrader)的基础上重新思考后编写的量化项目，其包含可扩展的数据recorder，api，因子计算，选股，回测，定位为**中低频** **多级别** **多标的** 全市场分析和交易框架。

# 使用方式
```
pip install -U zvt
```
